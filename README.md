# Elm NbE Tutorial

This repository contains an implementation of normalization by evaluation (NbE) in Elm.

## Examples

Examples are stored in the `examples/` directory. Each example is a self-contained exercise with instructions and optional starter code.

## How to use the examples

1. Open `examples/README.md` to see available exercises.
2. Read the example-specific `README.md` in the exercise folder.
3. Use the repository source files under `src/` to implement the feature.
4. Run tests or compile the code to verify your changes.

## Run tests

```bash
npx elm-test tests/IntegerTest.elm
```

For full project testing, run your Elm commands from the repository root.
