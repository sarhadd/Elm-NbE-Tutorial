# Examples

This folder contains small exercise prompts for the NbE tutorial.

## Available exercises

- `add-bool-literal/` — Add boolean literal support to the expression language
- `physical-units/` — Add a simple unit type for physics values

## How to use an example

1. Create a new git branch for the exercise, for example:
   `git checkout -b examples/add-bool-literal`
2. Open the example folder.
3. Read its `README.md` for the task description.
4. Use the snippet and the existing `src/` files to implement the feature.
5. Run Elm compilation or tests to verify the result.
